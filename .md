# Sistema para un Semáforo de Rendimiento Estudiantil

## Inicio

El presente proyecto tiene como objetivo desarrollar un Sistema para un Semáforo de Rendimiento Estudiantil para la Institución Educativa "ACADEMICSCOPE" de Cajamarca. Este sistema nace como respuesta a la necesidad de mejorar la comprensión de las calificaciones por parte de los padres de familia y fortalecer la comunicación entre docentes, estudiantes y tutores. Mediante una representación visual basada en colores, se busca facilitar el seguimiento del rendimiento académico y contribuir a una mejor toma de decisiones.

## Descripción

El sistema transforma las calificaciones del MINEDU (AD, A, B y C) en un semáforo de colores: verde, amarillo, naranja y rojo, permitiendo identificar de manera rápida el nivel de desempeño de cada estudiante. Para su desarrollo se emplearon tecnologías como Java/Spring Boot, HTML, CSS y herramientas de ingeniería de software, aplicando buenas prácticas de programación y metodologías de desarrollo para garantizar un sistema seguro, escalable y fácil de mantener.

## Conclusión

El proyecto permitió implementar una solución tecnológica que mejora el monitoreo del rendimiento académico mediante una interfaz clara e intuitiva. Asimismo, fortalece la comunicación entre docentes, estudiantes y padres de familia, facilitando la detección temprana de dificultades académicas. Gracias a las pruebas realizadas y a las buenas prácticas de desarrollo, se obtuvo un sistema funcional, confiable y preparado para futuras mejoras.
